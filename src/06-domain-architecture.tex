\section{Mandatory Infrastructure and Security Requirements}

\begin{enumerate}[leftmargin=*]
    \item All subdomains shall enforce HTTPS with managed TLS certificates and automatic renewal.
    \item `www.flytics.org` shall redirect non-canonical host variants to canonical HTTPS URLs.
    \item `api.flytics.org` shall enforce API authentication, rate limiting, request tracing, and versioned routing.
    \item `auth.flytics.org` shall be isolated with strict security headers, session protection, and MFA-ready flows.
    \item `files.flytics.org` shall serve only authorized content using signed URL policies and short-lived tokens.
    \item Internal consoles (`ops`, `admin`, `billing`) shall be protected with role-based access and strong authentication controls.
    \item `status.flytics.org` shall be logically isolated from core production systems to remain available during incidents.
    \item DNS records shall be centrally managed with change approval, audit history, and rollback capability.
\end{enumerate}
